# Check if a number is even

## 说明

Determines whether a given number is even

## 参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| num | `number` | - The number to check |

## 返回值

- 类型: `boolean`
- 说明: True if the number is even, false otherwise

## 示例

```ts
import { isEven } from './utils'
const result = isEven(4) // returns true
```
